# Phone Store – Fullstack Project

## Description
ระบบจัดการร้านขายโทรศัพท์มือถือ (Phone Store Management System)  
รองรับการจัดการสินค้า การขาย และผู้ใช้งาน  
ออกแบบมาเพื่อเป็นส่วนหนึ่งของ portfolio

---

## Tech Stack

### Frontend
- Next.js
- Tailwind CSS

### Backend
- Node.js
- Express
- Prisma
- MongoDB
- PostgreSQL (หรือฐานข้อมูลตามการตั้งค่า)

---

cd backend
